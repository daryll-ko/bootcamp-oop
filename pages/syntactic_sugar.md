## Syntactic sugar

<v-clicks>

**Syntactic sugar** allows us to write less code.

Like what?

- `x * x * x * x * x` vs `x ** 5`

<div class="flex justify-center gap-10 items-center">

```py
i = 0
while i < n:
	...
	i += 1
```

vs

```py
for i in range(n):
	...
```
